## Template Structure

The professional resume template used as reference follows a clean, ATS-optimized single-column layout with clear section hierarchy.

### Layout & Typography

**Overall Design:**
- Single-column, left-aligned layout
- Professional serif/sans-serif typography
- Generous whitespace for readability
- Clear section dividers (horizontal lines)
- Consistent spacing between sections

**Header Section:**
- Full name in large, bold text (24-28px)
- Contact information on second line: email, phone, LinkedIn, location
- All links clickable and properly formatted

**Section Headers:**
- All caps, bold, with underline separator
- Consistent spacing above and below
- Examples: PROFESSIONAL SUMMARY, TECHNICAL SKILLS, EXPERIENCE, PROJECTS, EDUCATION, CERTIFICATIONS, ACHIEVEMENTS

### Core Sections

**1. Professional Summary**
- 3-4 lines of concise overview
- Highlights key expertise, years of experience, and value proposition
- Action-oriented language

**2. Technical Skills**
- Organized in categories (Frontend, Backend, AI/ML, Databases, DevOps/Cloud, Payments/APIs, Tools)
- Each category lists relevant technologies separated by commas
- Skills presented in a clean table or list format

**3. Experience**
- Company name with role and date range (right-aligned)
- Bullet points describing achievements and responsibilities
- Quantifiable results where possible (e.g., "€1.2 Cr/month in volume")
- 3-5 bullets per role

**4. Projects**
- Project title with technologies used
- Brief description (1-2 lines)
- Key achievements and impact
- Links to live projects or repositories

**5. Education**
- Institution name, degree, field of study
- Graduation date
- GPA if notable (3.5+)
- Honors or distinctions

**6. Certifications**
- Certification name
- Issuing organization
- Date obtained

**7. Achievements**
- Key accomplishments and metrics
- Awards or recognition
- Quantifiable impact

### Color Scheme

**Primary Colors:**
- Text: Dark gray/black (#1a1a1a or #0f172a)
- Headers: Dark blue or black
- Links: Blue (#1e40af or similar)
- Dividers: Light gray or black

**Background:**
- White or off-white (#ffffff or #f8fafc)
- No background colors in sections

### Spacing & Margins

**Vertical Spacing:**
- Between sections: 12-16pt
- Between subsections: 8-10pt
- Between bullets: 4-6pt

**Horizontal Spacing:**
- Left/right margins: 0.5-0.75 inches
- Top/bottom margins: 0.5 inches
- Bullet indentation: 0.25 inches

### Font Specifications

**Primary Font:** Inter or similar sans-serif (modern, clean)
- Name: 24-28px, bold
- Section headers: 12px, bold, all caps
- Body text: 10-11px, regular
- Emphasis: Bold for company names, role titles

**Alternative:** Could use serif font for a more traditional look, but sans-serif is preferred for modern, tech-focused resumes.

---

## Key Design Principles

1. **ATS-Friendly:** Simple formatting, no images or complex layouts
2. **Scannable:** Clear hierarchy and section breaks for quick reading
3. **Professional:** Clean, modern aesthetic suitable for corporate and tech roles
4. **Comprehensive:** All standard resume sections represented
5. **Quantifiable:** Metrics and achievements highlighted
6. **Concise:** Efficient use of space, no unnecessary fluff

## Implementation Notes for HexaCv

### What to Include

- Clean, single-column layout matching this reference
- All seven standard sections (Header, Summary, Skills, Experience, Projects, Education, Certifications, Achievements)
- Real-time preview as user edits
- PDF export that preserves formatting
- Mobile-responsive design that stacks vertically on small screens
- Support for adding/removing sections and reordering

### What to Avoid

- Complex multi-column layouts
- Background colors or images
- Decorative elements that interfere with ATS scanning
- Inconsistent spacing or alignment
- Overly large fonts or unusual typography

### Responsive Considerations

**Desktop (1024px+):**
- Full single-column layout with proper margins
- Optimal readability at 8.5" x 11" paper size

**Tablet (641px - 1023px):**
- Slightly reduced margins
- Maintain single-column layout
- Adjust font sizes slightly for screen readability

**Mobile (320px - 640px):**
- Stack all content vertically
- Reduce margins to 0.25 inches
- Maintain section hierarchy and readability
- Ensure links are tappable (44px minimum height)

## Export & Print Specifications

**PDF Export:**
- Page size: 8.5" x 11" (Letter)
- Margins: 0.5" all sides
- DPI: 300 for print quality
- Font embedding: All fonts embedded
- Colors: Preserved as-is

**Print Optimization:**
- Ensure text is black or dark gray for print
- Links should be underlined or in a different color
- No background colors that consume ink
- Proper page breaks if resume extends to multiple pages
